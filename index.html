A unidade SESI - Sorocaba, proporciona para seus alunos e para alguns usuários (pagantes) diveras atividades, como estão citadas a cima.

Piscinas
Os usuários são livres para utilizarem a piscina quando quiserem, desde que façam o teste médico, que é feito no proprio clube gratuitamente

Quadras
Os usuários são livres para utilizarem a quadra, quando quiserem, uma das quadras é de futsal e a outra é de bsquete/vôlei
